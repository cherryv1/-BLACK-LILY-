# Lily Dashboard

Dashboard de administración para Lily Edge AI multi-modelo.

## Features

- ✅ Monitoreo en tiempo real (6 capas)
- ✅ Multi-modelo (Groq, Cerebras, Gemini, Cloudflare Llama)
- ✅ Métricas y analytics
- ✅ Logs con filtrado
- ✅ Alertas automáticas
- ✅ Reportes
- ✅ Análisis IA
- ✅ Control de usuarios
- ✅ Responsive

## Desarrollo

```bash
pnpm dev
```

## Tests

```bash
pnpm test
```

## Build

```bash
pnpm build
```

## Integración

Ver `src/worker-integration.ts` para integrar con Cloudflare Workers.
